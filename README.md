## 💡 The Idea

The project started with a simple question:

> What if a system could calculate future possibilities and recommend the best decision before a move is made?

Many AI systems, games, and planning engines evaluate future outcomes before choosing an action.

I wanted to visualize that process in a simple and interactive way.

Using Tic-Tac-Toe as the environment, Combo Explorer:

* 🔮 Calculates future possibilities
* 📊 Evaluates move quality
* 🔗 Predicts move chains
* 🎯 Recommends optimal actions

## ⚙️ How It Works

### 🌳 Path Enumeration

The engine explores all possible future game states and calculates:

* ❌ X Wins
* ⭕ O Wins
* 🤝 Draws

### 🧠 Minimax Algorithm

Assumes both players make optimal decisions.

The algorithm:

* ✅ Maximizes favorable outcomes
* 🛡️ Minimizes opponent advantages

### ✂️ Alpha-Beta Pruning

Reduces unnecessary calculations by skipping branches that cannot improve the final decision.

Result:

* ⚡ Faster analysis
* 📉 Fewer computations
* 🎯 Better performance
